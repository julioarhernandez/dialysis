# Global Dialysis Care Center

A modern web application built with **Astro** and **Tailwind CSS**.

## 🚀 Getting Started

To get the project up and running locally, follow these steps:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (LTS version recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd dialysis
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
The application will be available at [http://localhost:4321](http://localhost:4321).

### Build

Build the project for production:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```

## 🛠️ Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Integrations:** `@astrojs/tailwind`

## 📁 Project Structure

- `src/pages/`: Contains the routes/pages of the application.
- `src/components/`: Reusable UI components.
- `src/layouts/`: Common page layouts.
- `src/styles/`: Global styles and Tailwind configurations.
- `public/`: Static assets like images and fonts.
